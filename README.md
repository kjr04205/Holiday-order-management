### 프로젝트를 시작하게 된 이유
> 15년째 떡집을 운영하시는 부모님께서는 200~300명, 600되 정도의 주문량이 몰리는 명절에도 공책에 정보를 적어 운영을 하고 계십니다. 명절 전날에는 주문을 하신 손님들이 몰리게 되는데, 그때마다 한분한분 찾게 되니 많은 시간이 소요되었습니다. 효율적으로 관리를 하기 위해 프로젝트를 시작하였습니다.<br><br>
